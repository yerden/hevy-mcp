Exercise Logging Guidelines
---------------------------

---

**Workout title format:** Start with the primary muscle group names, then add a creative exclamation. Example: "Biceps & Triceps Annihilation!"

**Why:** User specified this style explicitly.

**How to apply:** Always follow this format when naming a workout before logging to Hevy.

---

**Unmatched exercises:** When an exercise doesn't have an exact match in Hevy, either:
- Use the most applicable Hevy exercise and add the distinguishing detail as a `notes` comment (e.g., "Bicep Curl (Dumbbell)" with note "Seated" for a seated variant), OR
- Advise the user to create a custom exercise if the movement is fundamentally different.

**Why:** User clarified this preference after the first log attempt.

**How to apply:** Before logging, flag any exercises that are approximate matches and state what note will be added. Only suggest a custom exercise if no reasonable built-in match exists.

---

**Exercise name mappings learned:**
- "Tricep pulldown with rope" → "Triceps Rope Pushdown" (`94B7239B`) in Hevy
- "Bicep curl with dumbbells (seated)" → "Bicep Curl (Dumbbell)" (`37FCC2BB`) + note "Seated"
